# DSA
My laboratory assignments and algorithmic problem-solving exercises for the DSA course at UBB.


This repository contains my coursework, laboratory assignments, and practical implementations for the Data Structures and Algorithms (DSA) course, part of the Computer Science B.Sc. program (German Study Line) at Babeș-Bolyai University.

The primary focus of these projects is to build a robust understanding of algorithmic thinking, memory management, and time/space complexity analysis (Big O). All implementations are built from scratch to solidify fundamental concepts before relying on standard libraries.
